Cholesky分解

对于有着特殊性质的矩阵，在这里我们考虑对称正定阵，我们可以设计一些奇妙的算法来对他进行分解

Cholesky分解的定理及证明

定理：Cholesky分解定理

设 ( A ) 是一个 ( n \times n ) 的对称正定矩阵，则存在一个唯一的下三角矩阵 ( L ) 使得：

$$
A = L L^T
$$

其中，( L ) 是一个下三角矩阵，其对角线上的元素均为正实数。

证明：

证明分为两部分：存在性和唯一性。

存在性：

由于矩阵 ( A ) 是对称正定的，我们知道它的所有特征值都是正实数。根据正定矩阵的性质，存在一个下三角矩阵 ( L )，使得$A = L L^T$。这一结果是 Cholesky 分解的基本结论。

唯一性：

设 ( L_1 ) 和 ( L_2 ) 都是满足 ( A = L_1 L_1^T ) 和 ( A = L_2 L_2^T ) 的下三角矩阵。如果我们假设 ( L_1 \neq L_2 )，则根据矩阵的正定性，矩阵 ( A ) 的每一行的元素都是由对应的下三角矩阵的元素线性组合得到的。通过计算可以得出，只有当 ( L_1 = L_2 ) 时，矩阵 ( A ) 的分解是唯一的。

因此，Cholesky分解是唯一的。
